# Sunmedia-Analytics

## Technology
#### [Vite](https://vitejs.dev/)
#### [MUI](https://mui.com/)
#### [i18next](https://www.i18next.com/)
#### [ReactRouter](https://v5.reactrouter.com/web/guides/quick-start)
#### [react-query](https://react-query-v3.tanstack.com/)

## Running frontend
### Install dependencies
```shell
yarn
```
### In development mode
```shell
yarn run dev
```
### Build project
```shell
yarn run build
```
### Preview project
```shell
yarn run preview
```

## Running husky
```shell
yarn run husky
```
